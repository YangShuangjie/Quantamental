# 简介

根据《基本面量化投资》（张然,汪荣飞 著，北京大学出版社）一书做的web应用。目前完成了基于mock数据的前端效果展示。

# 查看效果

```shell
git clone git@github.com:YangShuangjie/Quantamental.git
cd Quantamental
npm install
npm start
```
